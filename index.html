Project Overview

Speech sound disorders in children span four main areas: phonological disorders, articulation errors, fluency disorders (stuttering), and childhood apraxia of speech (CAS). Studies emphasize the need for early, personalized intervention using both evidence-based and AI-driven techniques. Phonological disorders involve rule-based sound errors and are effectively addressed through automated acoustic analysis and language-sensitive mobile apps. Articulation errors, caused by incorrect sound production, benefit from adaptive filtering and phoneme-specific practice tools. Fluency disorders like stuttering disrupt speech flow, with real-time AI feedback proving effective in promoting self-regulation. CAS affects motor planning and requires intensive, structured repetition. Deep learning models now enable precise detection and adaptive therapy for CAS. Together, research supports integrated, technology-enhanced systems like ArticuLearn that provide accessible, scalable, and multi-domain speech therapy tailored to each child’s needs.

Research Gap and Motivation

Lack of Integrated Multi-Disorder Platforms

Most tools address only one disorder, lacking a unified system for phonological, articulation, stuttering, and CAS therapies.

Motivation:

ArticuLearn offers an all-in-one solution with smooth transitions across therapy modules.

Limited Personalization in Therapy Delivery

Generic tasks are often used, without adjusting to a child’s evolving speech patterns or performance.

Motivation:

ArticuLearn personalizes content using LSTM classifiers and reinforcement learning algorithms.

Limited Clinical Validation

Many tools lack validation through real-user testing or pilot clinical studies.

Motivation:

ArticuLearn has undergone pilot testing, demonstrating both usability and therapeutic accuracy.

Research Problem & Solution

🧩
Research Problem

Children with speech sound disorders (SSDs) — including phonological disorders, articulation difficulties, fluency disorders (stuttering), and childhood apraxia of speech (CAS) — often struggle to access consistent, personalized, and effective therapy.

Existing solutions typically address only one disorder, lack real-time feedback, are not tailored to individual needs, and are rarely adapted for local languages or low-resource settings. These limitations hinder early diagnosis, delay progress, and reduce therapy engagement, especially in multilingual countries like Sri Lanka.

💡
Proposed Solution

ArticuLearn is an integrative, AI-powered mobile therapy system that combines advanced machine learning models with interactive tools to deliver personalized, real-time speech therapy for children aged 3–14.

The system includes specialized modules for phonology, articulation, stuttering, and CAS, each powered by techniques like LSTM networks, MFCC-based feature extraction, transformer attention mechanisms, and reinforcement learning.

It adapts to the child’s performance, provides instant feedback, supports local languages, and enables scalable, home-based therapy—bridging the accessibility and personalization gap in conventional systems.

Research Objectives

Unified AI-Based Therapy

Support for articulation, phonological, fluency, and CAS disorders in one platform.

Phoneme Error Detection

LSTM and deep learning using MFCCs to detect phoneme-level speech errors.

Adaptive Feedback Mechanism

Reinforcement learning to personalize therapy exercises dynamically.

CAS Deficit Detection

Use of attention models and DTW for motor planning deficit detection in CAS.

Mobile App Delivery

Interactive app for children aged 3–14 with multilingual support in low-resource areas.

System Evaluation

Test performance using normative and disorder-specific child speech samples.

Methodology

The methodology of the ArticuLearn project involves designing an integrated AI-based speech therapy system utilizing deep learning techniques such as LSTM for phoneme-level error detection, and reinforcement learning for personalized feedback. The system leverages Mel-Frequency Cepstral Coefficients (MFCCs) as key acoustic features to analyze speech signals. Attention mechanisms and dynamic time warping (DTW) algorithms are applied to detect and quantify motor planning deficits associated with childhood apraxia of speech (CAS). The therapy exercises are delivered through a user-friendly mobile application tailored for children aged 3 to 14, supporting multilingual access and accommodating low-resource settings. Pilot testing is conducted using both normative and disorder-specific child speech datasets to evaluate therapeutic impact, accuracy, and latency.
